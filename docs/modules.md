# Module Reference

## Core
Handles Safe Boot, dashboard, and environment checks.

## Corruption Engine
Performs integrity scans on Cache, DB, and DOM.

## Auto-Purifier
Repairs corrupted browser data.

## Safe Boot V2
Enhances staged boot system.

## Conversation Scanner
Checks JSON, Code Blocks, Unicode validity.

## Full Power Audit
Verifies CDN, routing, cache source.

## Version Checker
Auto-checks GitHub for updates.

## Debug Panel
Log/Warning/Error capture in real-time.
