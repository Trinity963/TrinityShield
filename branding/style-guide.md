# TrinityShield Visual Identity Guide

## Typography
- Primary: **Inter**
- Secondary: **JetBrains Mono** (code)

## Design Style
- Futuristic
- Neon purple cyber aesthetic
- Rounded edges
- Dark mode by default
- Light glow effects

## Icon Guidelines
- Centered shapes
- Soft inner glow (#BB86FF)
- Border strokes (#8E5CF6)
- Use triangles for "core" modules
- Use circles for “scanner/engine”
